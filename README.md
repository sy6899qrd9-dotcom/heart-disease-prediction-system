# heart-disease-prediction-system
The Heart Disease Prediction System is a machine learning-based web application developed using Python and Streamlit. It analyzes patient health parameters to predict the risk of heart disease in real time. The system provides accurate predictions, maintains patient records, and supports early diagnosis and preventive healthcare.
# 🫀 Heart Disease Prediction System

## Overview

The Heart Disease Prediction System is a machine learning-based web application developed using Python and Streamlit. The system predicts the likelihood of heart disease based on patient medical information and provides real-time risk assessment.

## Features

* Heart disease risk prediction using Machine Learning
* Interactive Streamlit user interface
* Real-time prediction and analysis
* Patient record management
* Reports download in CSV format
* User-friendly healthcare dashboard

## Technologies Used

* Python
* Streamlit
* Pandas
* NumPy
* Scikit-learn
* Machine Learning

## Input Parameters

* Age
* Gender
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Fasting Blood Sugar
* Resting ECG
* Maximum Heart Rate
* Exercise-Induced Angina
* ST Depression

## Installation

```bash
pip install -r requirements.txt
streamlit run interface.py
```
